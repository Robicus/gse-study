# Linux Command Line

## Basics

Clear Screen

```
CTRL + L
```

Current user

```
whoami
```

## Acount Stuff

Adding a new user

```
useradd -d [home] [login]
```

Changing Passwords

```
passwd [login_name]
```

## Filesystem Stuff + Navigation

Switching Directories

```
cd
```

List current directory


```
pwd
```

List directory contents:

```
ls
ls -la
```
